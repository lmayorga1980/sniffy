## Filter Network Traffic using Perl

### Requirements

* Ubuntu Linux with Perl 5.x
* `sudo apt install libnet-pcap-perl`
* `sudo apt install libnetpacket-perl`

### Run the Scripts

* `sudo perl sniffy.pl`
